# 🎮 Retro Reload – Online Game Store

## Project Overview
Retro Reload is an online game store system developed as a database project.  
The system allows users to browse retro games, manage accounts, and interact with the store platform.

The project demonstrates how database concepts and backend logic can be used to simulate a digital game marketplace.

## Features

User System
- User registration and login
- OTP authentication
- Account management

Admin System
- Admin control panel
- Manage users and games
- Monitor platform activity

Publisher Module
- Publishers can add and manage games

Game Store Interface
- Browse available games
- View game information
- Interactive store system

## Technologies Used

- Python
- SQL / Database concepts
- File handling
- GUI elements

## Project Structure

admin.py – admin management functions  
login.py – login authentication system  
otp.py – OTP verification module  
publisher.py – publisher features  
user.py – user system  
RR.py – main application file  

assets/ – contains project images and icons
